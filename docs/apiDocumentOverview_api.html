Overview

This page lists all of the namespaces exposed by the CVOM Visualization API, and the standard methods exposed by all visualizations. However, the API navigation panel on the left may not include all of the namespaces listed below, depends on the packaging options of this documentation.

Consumption API (sap.viz.api)

Consumption API includes methods help you to create various CVOM chart types via VizInstance() API. Besides, before creating a visualization instance, there are handy methods to set up data model and configure global charting options. Check out chart samples at Chart Gallery and Feature Demos.

Extension API (sap.viz.extapi)

Extension API is used when you want to create a CVOM extension. By using extension API, you can build a brand new visualization from scratch. Or you can also re-create a CVOM built-in chart via customization methods. More extension examples can be found at Extension Demos.
