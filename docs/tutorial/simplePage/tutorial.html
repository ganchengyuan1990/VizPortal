Overview

CVOM (Common Visualization Object Modeler) is a HTML5/JavaScript based visualization library. It includes more than 100 state-of-art business charts, geographic maps, a full-featured and highly extensible SDK, and, most importantly, a VizFrame service which makes integration of CVOM super easy!

As the pioneer and standard visualization provider in SAP, CVOM charts are fully enterprise ready, which even support keyboard navigation, screen reader and other advanced accessibility features. As a pure client and highly interactive library, CVOM is also tuned with Big Data. All CVOM charts can be fully used as you go. It is optimized on mobile device with gesture support, such as pinch & spread for zooming charts!

State-of-Art Charts

If you are the first user, you can experience the look & feel of all supported chart types at Chart Gallery. By reading Chart Concept, it will give you systematic understanding on how CVOM charts is design. Furthermore, when you want to learn how to use specific charting features, Feature Demos is a good place to try and learn. It gives you a live sample with full code snippet!

Each CVOM chart has hundreds of chart properties to configure the style or interactions behavior to meet users’ needs. If you’d like to customize chart, Chart Property Reference document should be the first place to look at.

CVOM charts are adopted and implemented in various SAP products like SAP Cloud for Planning, Lumira, Design Studio, Predictive Analysis and many other SAP Analytic tools. The subset of charts is also wrapped as sap.viz chart library in SAPUI5, and used by Fiori and hundreds of SAPUI5 applications, small or large. To understand more for how to use CVOM charts in SAPUI5, check out sap.viz.ui5.controls.VizFrame samples for detail.

CVOM supports customized packaging to only include needed chart types. It greatly helps loading performance, especially for cloud application. It also has separated packages for geographic maps and Hichert charts.

Open SDK

CVOM provides an easy-to-start and highly extensible SDK for customizing existing charts or creating brand new viz extension. It is truly build once, run anywhere in several of BI products without any additional code changes.

There were already hundreds of extensions created by SAP and SAP partners. You can go to SAP Github or SAP Analytic Extension Marketplace to find all available extensions. Most of extensions are free and open sources. They are good resources for ramping up when you want to build your own. Moreover, several extension samples for how to implement specific extension feature are also available at Extension Demos.

CVOM SDK doesn’t only include full-featured JavaScript APIs, but also provides an public and free IDE tool for developers (aka VizPacker), which makes extension creation super easy!

The official help document of VizPacker can be found at SAP HANA Cloud Documentation. CVOM SDK API documentation(aka. Visualization API Reference) is at API Documentation or SAP Help Portal.

To keep connected with our developer, join us in JAM Group: CVOM Visualization SDK and Extensions.

VizFrame

VizFrame provides several handy and lightweight services when you want to integrate CVOM into your applications or projects. It was previous known as VizControls or UVB (Universal Visualization Builder). Check out VizFrame for demos and VizFrame API in API Documentation for API usage.
