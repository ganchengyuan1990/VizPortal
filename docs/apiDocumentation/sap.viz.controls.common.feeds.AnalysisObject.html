iconData (iconData)

Get/Set AnalysisObject iconData
Parameters:
  • iconData : Array 
    •      [{
        "className" : "String", // Customer class name
        "tooltip" : "String", // (Optional) Set tooltip
        "position": "String" // Provide at most 3 icons position. "dataset", "hierarchy" are respectively put on text left, and "link" are put on text right.
     }]
    iconData allow user to customize className and tooltip.
Returns:
  • Array
Example:

Sample code:

var analysisObject = new AnalysisObject(...);
//Set custom icon font class
analysisObject.iconData = ([{
          "className" : "custom-class",
          "tooltip" : "this is a tooltip",
          "position" : "dataset"
     },{
          "className" : "custom-class",
          "tooltip" : "this is a tooltip",
          "position" : "hierarchy"
     }, {
          "className" : "custom-class",
          "tooltip" : "this is a tooltip",
          "position" : "link"
     }]);
var iconData = analysisObject.iconData();
