Constructor

SwitchBar (div, uiConfig)

The SwitchBar is a control that categories all chart types in groups. User can click on the arrow icon of group to display chart type items, click one of these items to change the current chart type.

Parameters:
  • div : HTMLElement
    • The dom element to host the control.
  • uiConfig : JSON [Optional]
    • Configuration for the control. If without the uiConfig, use the default uiconfig to create the control.
         uiConfig.visible:                 Show or hide the control, default value is true.
     uiConfig.defaultVizType:          Set the default viz type.
     uiConfig.groups:                  Configuration for categories of all viz types, each element of the groups
                                       is type of JSON Object, which contains "name" and "types".
    Sample Code:
       // Get a default uiConfig of SwitchBar from GlobalConfig. 
    var uiConfig = sap.viz.controls.common.config.GlobalConfig.defaultUIConfig(sap.viz.controls.common.config.GlobalConfig.DEFAULT_UICONFIG_TYPE_SWITCHBAR);
    // Customize the default uiconfig.
    uiConfig.defaultVizType = sap.viz.controls.common.constants.ChartConst.TYPE_LINE;
    uiConfig.groups = {
        "name" : LangManager.get("VIZ_SWITCHBAR_COLUMN_CHARTS"),
        "types" : [{
             "id" : ChartConst.TYPE_COLUMN
         }, {
             "id" : ChartConst.TYPE_STACKED_COLUMN
         }, {
           "id" : ChartConst.TYPE_DUAL_COLUMN
         }, {
          "id" : ChartConst.TYPE_3D_COLUMN
         }]
     }
    // Create a SwitchBar control.
    var switchBar = new SwitchBar($(div), uiConfig);

Events

vizTypeChanged

Dispatches "vizTypeChanged" event when visualization type changes. 
type: "vizTypeChanged"
handler: function({
                   type: "vizTypeChanged",
                   data: {
                       oldType: "",
                       newType: ""
                   }
})
Properties
  • evt.type : String
    • "vizTypeChanged"
  • evt.data.oldType : String
    • vizType
  • evt.data.newType : String
    • vizType
Example

Sample Code:

var switchBar = new SwitchBar(...);
var vizTypeChangedHandler = function(event) {
    var log = "vizTypeChanged event from switchBar and viz type changed from " +
                 event.data.oldType  + "to " + event.data.newType + ".";
    console.log(log);
};
switchBar.addEventListener('vizTypeChanged', vizTypeChangedHandler);

addEventListener (type, listener, scope, priority)

Registers an event listener object with an EventDispatcher object so that the listener receives notification of an event. You can register event listeners on any EventDispatcher object for a specific type of event, scope, and priority. If you no longer need an event listener, remove it by calling removeEventListener(), or memory problems could result.
Parameters:
  • type : String
    • The type of event.
  • listener : function
    • The listener function that processes the event.
  • scope : Object
    • The scope.
  • priority : int
    • The priority level of the event listener.
