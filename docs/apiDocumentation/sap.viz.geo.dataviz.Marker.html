Class: Marker

Class: Marker

Extends: sap.viz.geo.dataviz.LocationBasedViz

Constructor

Marker (options)

Create a new image marker visualization object. It will show images in geo locations specified by data.

Feeding Information: 

{
    location : FieldId1
}

Parameters:
  • options : Object [Optional]
    • the configuration object.
    Properties
    • url : String [Optional]
      • the URL of image. default value indicates by Path("sap.viz.map.Resources") + "/images/marker.png". Path could be set by sap.viz.api.env.Resource.path("sap.viz.map.Resources", "../../resources/data/geo/");
    • highlightUrl : String [Optional]
      • the URL of highlight image. default value = Path("sap.viz.map.Resources") + "/images/marker_highlight.png". Path could be set by sap.viz.api.env.Resource.path("sap.viz.map.Resources", "../../resources/data/geo/");
    • width : Number [Optional]
      • the width of image. default value = 12.
    • height : Number [Optional]
      • the height of image. default value = 18.
    • xoffset : Number [Optional]
      • the offset in horizontal. default value = -6.
    • yoffset : Number [Optional]
      • the offset in vertical. default value = -8.
    • cluster : Object [Optional]
      • the configuration object for cluster.
      Properties
      • enabled : Boolean [Optional]
        • whether enable clustering . default value = false.
      • url : String [Optional]
        • the URL of image. default value indicates by Path("sap.viz.map.Resources") + "/images/cluster.png". Path could be set by sap.viz.api.env.Resource.path("sap.viz.map.Resources", "../../resources/data/geo/");
      • highlightUrl : String [Optional]
        • the URL of image. default value indicates by Path("sap.viz.map.Resources") + "/images/cluster_highlight.png". Path could be set by sap.viz.api.env.Resource.path("sap.viz.map.Resources", "../../resources/data/geo/");
      • xmargin : Number [Optional]
        • the distance between image border and label in horizontal. default value = 15.
      • ymargin : Number [Optional]
        • the distance between image border and label in vertical. default value = 15.
      • minPts : Number [Optional]
        • min points in cluster. default value = 3.
      • distance.maxDistance : Number [Optional]
        • the max distance(num of pixels), 80 by default.
      • distance.minDistance : Number [Optional]
        • the min distance(num of pixels), 20 by default.
    • dataLabel : Object [Optional]
      • the configuration object for data label.
      Properties
      • visible : Boolean [Optional]
        • whether enable data labels, false by default.
