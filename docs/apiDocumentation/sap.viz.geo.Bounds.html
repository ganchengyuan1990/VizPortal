Constructor

Bounds (arg0, arg1)

Create a bounding box object.
Parameters:
  • arg0 : Array | Number | sap.viz.geo.Bounds [Optional]
    • If it is
    • a number, it means the initial dimension of the bounding box. If not defined, the default dimension is 2.
    • a "sap.viz.geo.Bounds" object, it will create a copy of this bounds object.
    • an array and there's only one parameter, it should have 2*dimension numbers which are the coordinate values of two vertexes. And if the second parameter exists, this array will be treated as one of vertexes.
  • arg1 : Array [Optional]
    • the coordinate of the second vertex (longitude and latitude)
