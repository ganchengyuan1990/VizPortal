[static] enableDraggableFeeding (dom, analysisObject, backupAOs)

Enable a jqueryui draggable element to drop to viz controls, such as FeedingPanel.
Parameters:
  • dom : HTMLElement
    • A jqueryui draggable dom element
  • analysisObject : sap.viz.controls.common.feeds.AnalysisObject
    • An analysis object will be added to analysis container when drop.
  • backupAOs : Array<sap.viz.controls.common.feeds.AnalysisObject>
    • optional, when analysisObject is not valid, backupAOs will be checked one by one till some element is valid.
Example:

Sample Code:

var draggable$ = $('#divAppDraggable');
draggable$.draggable();
DragManager.enableDraggableFeeding(draggable$[0], analysisObject);

[static] disableDraggableFeeding (dom)

Disable the draggable element to drop to viz controls.
Parameters:
  • dom : HTMLElement
    • The dom element which is enableDraggableFeeding
Example:

Sample Code:

var draggable$ = $('#divAppDraggable');
draggable$.draggable();
DragManager.enableDraggableFeeding(draggable$[0], analysisObject);
DragManager.disableDraggableFeeding(draggable$[0]);

[static] getDefaultHelper (dom, options)

Get a default helper element to be used for dragging display.
Parameters:
  • dom : HTMLElement
    • The dom element which is enableDraggableFeeding
  • options : JSON
    • optional, customize className for icon. Suggest to add "!important" to make sure customized styles will overwrite the default styles. If without the options, use the default uiconfig.
         optional options: "className" : "custom-class"
Returns:
  • HTMLElement
    • helper
Example:

Sample Code:

var draggable$ = $('#divAppDraggable');
var options = {"className" : "custom-class"};
draggable$.draggable({
    'helper' : function(event) {
        return DragManager.getDefaultHelper(this, options);
    }
});
DragManager.enableDraggableFeeding(draggable$[0], analysisObject);
