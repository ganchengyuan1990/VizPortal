Namespace: BaseCustomRenderer

The base object of custom render , static function can be overridden.

Properties

[static] id

Define id of BaseCustomRenderer

[static] type

Define type of BaseCustomRenderer, validated value can refer to CustomRendererTypes.

[static] metadata

Set the metadata of custom renderer of chart, it includes propertyDefinition.
propertyDefinition : Object
propertyDefinition schema: {
                  "supportedValueType": "Object", // Number, String, Boolean, StringArray, NumberArray, BooleanArray
                  "defaultValue": null,
                  // optional, by default read-only is false
                  "readonly": false,
                  // optional, by default serializable is true
                  "serializable": true,
                  // optional
                  "description": "",
              }

[static] render (ctx, properties, graphicInfo, vizModelInfo)

Delegate render function of chart module, it could be the render function of the following chart properties: 
        categoryAxis.labelRenderer
        plotArea.markerRenderer
        plotArea.dataLabel.renderer
Parameters:
  • ctx : Object
    • ctx: the data context of chart module.
    Sample: 
        {
         "Year": "2001",
         "Country": "China",
         "Product": "Moto"
    }
  • properties : Object
    • properties: the custom render setting for specified ctx.
    Sample: 
        {
         fill: "blue",
         stroke: "blue"
    }
  • graphicInfo : Object
    • graphicInfo: the graphic information from the chart.
    supported renderers: 
        categoryAxis.labelRenderer
    plotArea.markerRenderer
    plotArea.dataLabel.renderer
   
     Sample:
 
     GraphicInfo Of "categoryAxis.labelRenderer": 
     {
    
         //   cell position in the current axis layer
        'position':{
             //   x: the cell x position
            "x": 30,
             //   y: the cell x position
            "y": 0,
        },
        
         //   size: the actual size of the cell could be renderered. it's already calcuated by the oritation,
        "size": {
            "width": 121.69039062000002,
            "height": 109.76
        },
         //   text: the label text
        "text": "USA",
         //   style: the label style which set in the chart properties
        "styles": {
            "color": "#333333",
            "fontFamily": "'Open Sans', Arial, Helvetica, sans-serif",
            "fontSize": "12px",
            "fontStyle": "normal",
            "fontWeight": "normal"
        },
   
     },
    
 GraphicInfo Of "plotArea.markerRenderer": 
     {
    
        "graphic": {
             // size: lineMarker gets NO size property.
            'size':{
                 // width: the data point width
                "width": 34,
                 // height: the data point height
                "height": 116.88622734375001,
            }
             // fill: the color of marker.
            "fill": "#8873A2",
             // totalStackHeight: totalStackHeight: the current total stack height, its specifies for stacked based chart.
            'totalStackHeight': 100,
             // stackPosition: the stacked marker position in current category, its specifies for stacked based chart.
            'stackPosition':{
                 // x: the x positon which layouts in current category, it specifies for stacked based chart.
                 //    the start point starts<0>.
                "x": 0,
 
                 // y: the y positon which layouts in current category, it specifies for stacked based chart.
                 //    the start point starts<0>.
                "y": 142.6281035958904,
            }
        },
         // horizontal: the chart orientation.
        "horizontal": false,
         // combinationIsLineMarker: specify for combination chart, illustrate the marker is line marker or column/bar maker.
        "combinationIsLineMarker": false,
    
    },
    
 GraphicInfo of "plotArea.dataLabel.renderer": 
    {
    
         // value: the row value of label
        "value": 369,
         // text: the text of label
        "text": "369",
         // styles: the style which set in the chart property 
        "styles": {
            "color": "#333333",
            "fontFamily": "'Open Sans', Arial, Helvetica, sans-serif",
            "fontSize": "12px",
            "fontWeight": "normal",
            "fontStyle": "normal",
        },
        // dataPointSize: the size of current data point
        "dataPointSize": {
            'width': 200,
            'height': 300
        }
    
    }
  • vizModelInfo : Object
    • vizModelInfo: the vizModelInfo of current customization chart,
    detailed structure can be referred from vizModelInfo.
