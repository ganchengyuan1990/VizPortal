Parameters:
  • options : Object [Optional]
    • the configuration object.
    Properties
    • data : Object [Optional]
      • an array holding data to be shown in this layer.
    • viz : Object [Optional]
      • a data visualization.
    • feeding : Object [Optional]
      • a object of feeding.
    • legendVisible : Boolean [Optional]
      • a boolean value to indicate whether the legend of this layer should be shown. Default is false.
    • tooltip : Object [Optional]
      • the configuration of tooltip.
      Properties
      • formatString : String | Object [Optional]
        • a string or a object which is the format pattern of numbers in tooltip. It can be also an object keys of which are ids of measures to specify the patterns separately.
      • unitFormatType : String [Optional]
        • a string which is the type of format pattern of numbers in tooltip. It can be also an object keys of which are ids of measures to specify the patterns separately.
