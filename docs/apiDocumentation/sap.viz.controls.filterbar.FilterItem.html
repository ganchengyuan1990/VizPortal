Class: FilterItem

Constructor

FilterItem (id, displayName, displayValue, customData, cssClass, valueCssClass, disableRemoval, valueCount, iconData)

The FilterItem class is the UI abstraction of a visual Filter.
Parameters:
  • id : String
    • filter item id
  • displayName : String
    • filter item display name
  • displayValue : String
    • filter item display value
  • customData : Object [Optional]
    • optional, custom data for this filter item
  • cssClass : String [Optional]
    • optional, css class name of the FilterRenderer's base DIV to support different styles like background color
  • valueCssClass : String [Optional]
    • optional, css class name of the display value
  • disableRemoval : Boolean [Optional]
    • optional, disable removal icon, default value is false
  • valueCount : Number [Optional]
    • optional, value count in filter, default value is 0
  • iconData : JSON [Optional]
    • optional, set customize className and tooltip
