Class: Map

Extends: sap.viz.geo.EventSource

Constructor

Map (container, options)

Create a new geo map object. Event "mapInitialized" will be fired when the map is created and rendered.
Parameters:
  • container : HTMLElement
    • the DOM element to place geo map
  • options : Object [Optional]
    • the geo map initial configuration:
    Properties
    • baseMap : sap.viz.geo.basemap.AbstractProvider [Optional]
      • Provide a base map provider object which should be an implementation of AbstractProvider.
    • selectionModel : Object [Optional]
      • Settings of the selectionModel.
      Properties
      • mode : string [Optional] [Default: "inclusive"]
        • The selection mode to set.
    • tooltip : Object [Optional]
      • the options of tooltip
    • mapControls : boolean [Optional] [Default: true]
      • whether to show the map control which is a group of control buttons including zoom in/out, reset and lasso enabling
    • basemapSwitcher : boolean [Optional] [Default: true]
      • whether to show the basemap switcher
    • template : string [Optional]
      • the id of template to apply.

Events

mapInitialized

Fired when the map is initialized.

zoomComplete

Fired when zooming on map is finished.

panComplete

Fired when panning on map is finished.

resizeComplete

Fired when resizing on map is finished.
