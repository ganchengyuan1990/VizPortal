Fully Customizable Visualization

CVOM Standard Visualization

This simple and beautiful charting library includes over 100 charts, from basic lines, bars, etc., to maps, Hichert business charts, and more. All of these charts work across desktop browsers and mobile devices. CVOM is the standard charting library at SAP, widely used in many different applications and lines of business.

Design Beautiful Visualizations

CVOM’s flexible API allows you customize and control all of the details of your visualization, such as fonts, colors, and layout options. If you need to go beyond the standard charts and develop custom visualizations to fit your business case, CVOM’s powerful SDK makes it easy for you to build and deploy these in multiple SAP applications.

Same Delight from Desktop to Device

Our visualizations are designed to be “build once, run anywhere”. The same chart can be rendered on desktop and mobile devices, while our responsive chart layout optimizes the chart display for each platform. You can present your data consistently on different devices without any extra development.

Open SDK

CVOM provides an easy-to-start and highly extensible SDK for customizing existing charts or creating brand new viz extension. SDK doesn't only include full-featured JavaScript APIs, but also provides a IDE tool for developers (aka "VizPacker") to make extension creation super easy! Once build your own viz extensions, it can seamlessly be integrated across different SAP products and expanding its value. The supported products includes SAP Lumira, Design Studio, SAPUI5 and the list is increasing.
