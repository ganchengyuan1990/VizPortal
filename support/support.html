Download

CVOM follows up SAP lean DI development process, both CVOM artifacts and documentation are managed in Nexus.

To download latest CVOM, access Nexus with following link

https://nexus.wdf.sap.corp:8443/nexus/index.html#nexus-search;gav~com.sap.viz~sap.viz.charts

There are several packages offering (zip) is listed below, unzip it and you are ready to go.

  • sap.viz.charts-<version>-main.zip         - Standard CVOM charting library with SDK
  • sap.viz.charts-<version>-sdk.zip           - CVOM SDK library, excludes any built-in chart types
  • sap.viz.charts-<version>-hichert.zip      - Hichert style charts built by CVOM
  • sap.viz.charts-<version>-sapui.zip        - Optimized CVOM charting library integrated with in SAPUI5
  • sap.viz.charts-<version>-fpa.zip           - CVOM charting library for SAP Cloud for Planning
  • sap.viz.charts-<version>-lumira.zip       - CVOM charting library for SAP Lumira

NOTE: replace <version> to exact version you want to download, such as sap.viz.charts-5.19.0-main.zip


Release Information

Release notes are included in each CVOM package (zip) file for every release. If you want to follow up our latest update, go and subscribe CVOM Update JAM Group
